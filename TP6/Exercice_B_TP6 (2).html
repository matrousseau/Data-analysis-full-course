def build_diff(df):
  
  diff = np.zeros(df.shape[0]-1)
  
  for i in range (1,len(diff)):
    
    diff[i-1] = df.iloc[i]-df.iloc[i-1]
    

  return diff

X = DiffPNB.values

def test_model(p,q):

  size = int(len(X) * 0.66)
  train, test = X[0:size], X[size:len(X)]
  history = [x for x in train]
  predictions = list()
  for t in range(len(test)):
    model = ARMA(history, order=(p,q))
    model_fit = model.fit(disp=0)
    output = model_fit.forecast()
    yhat = output[0]
    predictions.append(yhat)
    obs = test[t]
    history.append(obs)
  error = mean_squared_error(test, predictions)
  
  print(model_fit.summary())
  plt.plot(test)
  plt.plot(predictions, color='red')
  plt.show()


  return model_fit

On utilise le log pour simplifier les calculs.
  • Akaike's Information Criterion : mesure de la qualité d'un modèle statistique. Plus il est bas, mieux est le modèle

Pour deux des trois modèles ,on peut accepter l'hypothèse de la distribution normale de la série au vue des résultats du test de Shapiro ((8,2) et (0,2)).

En ce qui concerne le test de Box-Pierce pour ces deux modèles les p-values sont vraiment très proche de 1. La répartition des résidus est aléatoire donc on peut considérer qu'il y a une stationnarité.

Notre dernier modèe (8,2) possède l’autocorrélation pour les différents décalages la plus faible ce qui nous amène à conclure que c'est le modèle le plus pertinent des trois.
